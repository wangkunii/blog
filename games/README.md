# 闯关游戏

本文件夹包含7个门徒训练闯关游戏，每个游戏都是独立的网页应用。

## 文件夹结构

```
games/
├── game1/           # 游戏1：以生命影响生命
│   ├── index.html   # 主页面
│   ├── style.css    # 样式文件
│   ├── game.js      # 游戏逻辑
│   ├── questions.js # 题目数据
│   └── sounds/      # 音效文件夹
│
├── game2/           # 游戏2：初信栽培系列
│   ├── index.html
│   ├── style.css
│   ├── game.js
│   └── questions.js
│
├── game3/           # 游戏3：信徒培训课程
├── game4/           # 游戏4：教会完整观
├── game5/           # 游戏5：查经法
├── game6/           # 游戏6：讲道学
├── game7/           # 游戏7：祷告服侍
│
└── README.md        # 本说明文件
```

## 游戏列表

1. **游戏1：以生命影响生命** - 基础门徒训练知识
2. **游戏2：初信栽培系列** - 初信者成长指南
3. **游戏3：信徒培训课程** - 系统神学基础
4. **游戏4：教会完整观** - 教会论入门
5. **游戏5：查经法** - 读经方法与技巧
6. **游戏6：讲道学** - 讲道基础训练
7. **游戏7：祷告服侍** - 祷告生活操练

## 运行方式

直接在浏览器中打开对应游戏的 `index.html` 文件即可开始游戏。

## 游戏特点

- 每个游戏包含10道选择题
- 即时反馈答题结果
- 进度条显示当前进度
- 完成后显示得分和鼓励语
- 支持重新开始功能
- 响应式设计，支持移动端
- 每个游戏使用不同的渐变配色方案
