# 以生命影响生命 - 门徒训练闯关游戏部署指南

## 项目结构
```
discipleship_quiz/
├── index.html      # 游戏主页面
├── style.css       # 样式文件
├── game.js         # 游戏逻辑
└── questions.js    # 题库（60道题）
```

## 部署方式

### 方法1：本地直接打开（最简单）
1. 将整个 `discipleship_quiz` 文件夹复制到手机
2. 在手机上使用浏览器打开 `index.html` 文件
3. 游戏会自动加载，无需网络连接

### 方法2：本地服务器（推荐）
1. 在电脑上启动本地服务器：
   ```bash
   cd discipleship_quiz
   python -m http.server 8082
   ```
2. 确保手机和电脑在同一WiFi网络下
3. 在手机浏览器中访问：`http://电脑IP:8082`
4. 例如：`http://192.168.1.100:8082`

### 方法3：部署到GitHub Pages
1. 将项目上传到GitHub仓库
2. 在仓库设置中开启GitHub Pages
3. 选择 `main` 分支和根目录
4. 获取生成的URL并分享给他人

### 方法4：部署到Netlify
1. 访问 https://app.netlify.com/
2. 点击 "Add new site" → "Deploy manually"
3. 拖放 `discipleship_quiz` 文件夹
4. 等待部署完成，获取URL

## 技术特性

### 纯静态文件
- 所有游戏逻辑在客户端运行
- 无需后端服务器
- 支持离线使用
- 本地存储排行榜数据

### 响应式设计
- 完美适配手机、平板、电脑
- 支持触控操作
- 自动调整布局

### 分享功能
- 生成的URL可以直接分享给朋友
- 朋友打开链接即可开始游戏
- 无需安装任何应用

## 游戏特性
- 📚 基于《以生命影响生命的门徒训练》12课教材
- ❓ 每次随机生成20道题目
- ⭐ 每题10分，满分200分
- 🎯 合理的干扰项增加挑战性
- 🏆 详细的评分系统和徽章
- 📊 答题回顾和解析
- 👥 支持多人同时游戏
